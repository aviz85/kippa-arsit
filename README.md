# כיפה ערסית (Kippa Arsit)

הרפתקה גרפית בסגנון סיירה (SCI, סוף שנות ה-80) — קומדיה ישראלית מבוססת על "כיפה אדומה".
הפעם **כיפה** היא ערסית של השכונה, **הזאב** צמחוני, ביישן ונשמה, ו**הסבתא** היא הנבל האמיתי: פסיכופתית מתוקה עם אוסף סכינים. סל אחד, יער אחד, וטוויסט אחד גדול.

נכתב ב-Vanilla JS + HTML5 Canvas, **בלי שום שלב build**. רזולוציה לוגית 320×200, מוגדלת לפיקסל-ארט EGA.

---

## איך מריצים

המשחק טוען קבצי JS דרך `<script>`, אז צריך לשרת אותו דרך שרת מקומי (פתיחת `index.html` ישירות מהקובץ לא תעבוד בגלל הגבלות הדפדפן):

```bash
cd <תיקיית הפרויקט>
python3 -m http.server
```

ואז פותחים בדפדפן:

```
http://localhost:8000/index.html
```

לוחצים על **"▶ התחל משחק"** (או Enter / רווח) ויוצאים לדרך.

## שליטה

- **בוחרים פעולה** בסרגל הפעלים: **ללכת / להביט / לדבר / לקחת / להשתמש**.
- **לוחצים על המסך** או על אובייקט כדי להפעיל את הפעולה שנבחרה עליו.
- **להשתמש בחפץ**: בוחרים את הפעולה "להשתמש", בוחרים חפץ מהאינוונטורי, ואז לוחצים על המטרה במסך (למשל: להשתמש ב**גרעינים** על **הזאב**).
- **Tab** — מעבר לפעולה הבאה.
- **רווח / לחיצה** — להמשיך טקסט (שורת ההודעה הלבנה) ולעבור לשורה הבאה.
- **F5** — שמירה · **F7** — טעינה.

טיפ בסגנון סיירה: זה משחק שאפשר למות בו (בצורה מצחיקה). שמרו לעיתים קרובות.

## על המשחק

מסע של תחנה אחר תחנה: מהחדר של כיפה, דרך הסלון והשכונה, אל היער — שם פוגשים את הזאב הצמחוני, מתקנים גשר שבור, ומגיעים אל בית הסבתא. המטרה: להגיע לעימות הסופי **כשהזאב כבר חבר ויש בידיים הוכחה** נגד הסבתא. בלי שניהם — הסוף ארור.

הטון: קומדיית רחוב ישראלית, חמה ומצחיקה — לא גסה ולא מרושעת.

הדרך המלאה לניצחון: ראו `docs/WALKTHROUGH.md`.

---

## מבנה הקוד

- `index.html` — סדר הטעינה של כל הסקריפטים.
- `js/engine.js` — המנוע: ציור פיקסל-ארט, סצנות, אינוונטורי, דיאלוג, מוות/ניצחון.
- `js/sprites.js`, `js/npc/*.js` — דמות השחקנית (`red`) ושאר ה-NPCs.
- `js/items.js` — איקוני החפצים באינוונטורי.
- `js/story.js` — קולות הדמויות והדגלים (single source of truth).
- `js/scenes/scene00..scene10` — הסצנות, מהאינטרו ועד העימות.
- `docs/` — מסמכי התכן: ספר הסיפור, ספר האמנות, גרף הפאזלים, ועוד.
